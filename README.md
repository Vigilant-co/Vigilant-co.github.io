# Vigilant Organization Website

Professional website for Vigilant development organization.

## Features
- Dark theme with gold accents
- Fully responsive design
- Four main pages: Home, Team, Projects, Contact
- Interactive elements with CSS/JS
- Optimized for GitHub Pages deployment

## Pages
1. **Home** - Introduction, values, current focus
2. **Team** - Team members, roles, skills
3. **Projects** - NimbleTrace showcase, roadmap
4. **Contact** - Contact form, FAQ, links

## Technology
- Pure HTML5, CSS3, JavaScript
- No frameworks or build tools
- CSS Grid & Flexbox layout
- CSS Custom Properties for theming

## Deployment
1. Push to GitHub repository named `vigilant-co.github.io`
2. Enable GitHub Pages in repository settings
3. Site will be available at `https://vigilant-co.github.io`

## Local Development
Open `index.html` in any browser or use:
```bash
    python3 -m http.server 8000
```
