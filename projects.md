# Our Projects

## NimbleTrace
**Status:** Active Development  
**Type:** Price Monitoring & Tracking System  
**Repository:** [Vigilant-co/nimble-trace-core](https://github.com/Vigilant-co/nimble-trace-core)

### Overview
NimbleTrace is a professional-grade price tracking system designed for developers, analysts, and anyone needing to monitor product prices across multiple platforms.

### Core Features
- **Real-time Price Tracking** - Live monitoring across websites
- **Multi-source Support** - Amazon, Digikala, eBay, and more
- **Automated Alerts** - Price drop/rise notifications
- **Data Visualization** - Interactive charts and dashboards
- **Historical Analysis** - Price trend tracking over time

### Technology Stack
**Backend:** Go (Gorilla Mux), PostgreSQL  
**Scraping:** Python (BeautifulSoup, Requests)  
**Frontend:** HTML/CSS/JS, Chart.js  
**DevOps:** Docker, GitHub Actions
